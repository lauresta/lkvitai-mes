## Žurnaliniai įrašai (Audit Logs)

Sistemos audito žurnalas pasiekiamas per:

- Meniu: `Admin` → `Audit Logs` (UI: Admin → Audit Logs)
- Maršrutas: `/warehouse/admin/audit-logs`

Administratorius gali matyti visus sistemos veiksmus su data, vartotoju ir IP adresu.
