# Operator Training Guide (Phase 1.5)

## Training Modules

1. Receiving and QC baseline workflows.
2. Wave picking execution.
3. Cross-docking routing.
4. RMA intake and disposition.
5. Analytics interpretation (fulfillment + quality).

## Suggested Session Plan

- Session 1 (30 min): Core navigation and auth.
- Session 2 (45 min): Wave picking + dispatch dependencies.
- Session 3 (30 min): RMAs and defect handling.
- Session 4 (20 min): KPI dashboards and escalation process.

## Training Materials

- Scripted walkthroughs in Blazor routes.
- Screenshot-based quick references (to be captured in environment).
- Recording checklist for each module:
  - start state
  - steps
  - expected outcomes
  - known failure modes

## Completion Criteria

- Operator completes a wave from create -> complete without intervention.
- Operator records cross-dock completion.
- Operator creates, receives, and inspects one RMA.
- Operator identifies late shipment root cause from analytics page.
