## Rules/Assumptions
- API source: all controller endpoints under `src/LKvitai.MES.Api/Api/Controllers/*.cs` plus minimal API mappings in `src/LKvitai.MES.Api/Program.cs`.
- UI source: Blazor routes (`@page`), nav menu entries, and page actions calling typed clients in `src/LKvitai.MES.WebUI/Services/*Client.cs`.
- `COVERED_INDIRECTLY` means route-matched workflow usage where the endpoint is not the page primary action.
- `INTENTIONAL_NO_UI` limited to security, observability, or maintenance APIs with explicit justification keywords.
- Remaining unmapped business/operational APIs are classified `GAP_NO_UI`.
